### 微信公众号(订阅号,服务号)

### 配置管理
* 微信配置文件
```
resources/wechat.properties
```
* MchId 商户号

* MchSecret 商户密钥(32位的)

* Domain 微信支付通知地址
例子
```
http://zhongyu.ren/pay_notify
```
